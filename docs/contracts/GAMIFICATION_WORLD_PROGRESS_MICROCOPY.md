# GAMIFICATION_WORLD_PROGRESS_MICROCOPY

Fecha: 2026-03-01  
Estado: Draft v1 (primer esfuerzo)
Scope: CountriesSheet (KPI `% del mundo`)

## Objetivo

Definir micro-copys de progreso para reforzar motivación en Explore sin convertir la UI en juego explícito.

## Principios

- El KPI `% del mundo` es informativo (no accionable).
- La motivación debe sentirse ligera y útil, no infantil.
- El copy debe ser corto para no romper layout móvil.

## Tramos de progreso (v1)

- `< 10%`: `Primeros pasos`
- `10% - 29%`: `Buen ritmo`
- `30% - 59%`: `Gran avance`
- `>= 60%`: `Nivel explorador`

## Implementación inicial

- Mostrar una barra de progreso (0–100) dentro del KPI de `% del mundo`.
- Mostrar micro-copy debajo de la barra según tramo.
- Mantener el valor numérico `%` como señal principal.

## Próximos pasos sugeridos

- Test A/B de copy por tramo.
- Evaluar si mostrar también `X/195 países` junto al porcentaje.
- Revisión de tono con idioma/locale.
