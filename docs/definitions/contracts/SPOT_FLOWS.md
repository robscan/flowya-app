# Spot Create/Edit Flows — Contract

## Create Spot (2-phase)

- **Phase 1 — Draft**: Create spot with only `{location, cover_image_url}`. No text fields.
- **Phase 2 — Edit texts**: Edit title, short_desc, long_desc, tags, etc. in full-screen screen only.

## Explore interaction

- Sheet is for quick actions + summary.
- Long text editing never happens inside the sheet.

## Navigation

- "Editar detalles" always opens a full-screen screen (no modal).
